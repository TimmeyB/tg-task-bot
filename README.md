# Task Bot

Telegram bot that pays users for completing tasks, with admin approval.

## Setup
1. Message @BotFather on Telegram, create a bot, copy the token
2. Message @userinfobot to get your numeric Telegram ID
3. Set BOT_TOKEN and ADMIN_IDS as environment variables when deploying
4. Deploy on Railway.app or Render.com connected to this repo

## Commands
User: /start /tasks /balance /withdraw
Admin: /addtask /pending /withdrawals
